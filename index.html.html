🚀 Enable Google Drive Cloud Sync

To enable automatic cloud sync across all your devices:

📋 Quick Setup (Recommended):

  1. Download and save this HTML file to your computer
  2. Open Command Prompt/Terminal:
    • Windows: Press Win + R, type cmd, press Enter
    • Mac: Press Cmd + Space, type "Terminal", press Enter
    • Linux: Press Ctrl + Alt + T
  3. Navigate to the file location:
    cd Desktop
    (or wherever you saved the file)
  4. Start the server:
    python -m http.server 8080
    or if that doesn't work:
    python3 -m http.server 8080
  5. Open in browser:
    http://localhost:8080
  6. Click your HTML file and then "☁️ Connect to Google Drive"
  7. Sign in to Google and grant permissions
  8. Done! Your data will automatically sync across devices

💡 Why do I need this?

Google's security requires a proper web server (not file://) for OAuth authentication. This is a one-time setup that enables secure cloud sync.

🔧 Alternative: Using VS Code (if you have it)

  1. Open the HTML file in VS Code
  2. Install "Live Server" extension
  3. Right-click the file → "Open with Live Server"
  4. Enable cloud sync in the opened page
